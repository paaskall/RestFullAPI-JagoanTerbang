API Documentation

Endpoint 1: GET /shipping_cost

This endpoint is used to retrieve the shipping cost based on the origin, destination, and weight.

  • Method: GET
  • Authentication: API Key required

Parameters

  • origin (required) - Origin city code
  • destination (required) - Destination city code
  • weight (required) - Package weight in kilograms

Response Format

{
    "status": "success",
    "data": {
        "cost": 50000,
        "estimated_days": 3
    }
}

Usage Examples

Example Request

GET /shipping_cost?origin=JKT&destination=BDO&weight=5

Example Response

{
    "status": "success",
    "data": {
        "cost": 50000,
        "estimated_days": 3
    }
}
